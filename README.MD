# INSTALLATION

Install easy-cron through node package manager:

```npm install easy-cron```

# USAGE

Using easy-cron is simple:

```
var easyCron = require('easy-cron');

var cronPattern = easyCron('Every five minutes'); // returns '*/5 * * * *'
```
